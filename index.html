DeautherX

Прошивка ESP8266 прямо из браузера через Web Serial. Нужен Chrome или Edge на компьютере.

1 · Тип дисплея

Браузер не поддерживает Web Serial. Открой страницу в Chrome или Edge на компьютере (Safari и мобильные браузеры не подойдут).

Как прошить

  1. Выбери тип дисплея (по умолчанию 1.3″, можно сменить).
  2. Подключи плату по USB и нажми «Подключить и прошить».
  3. В диалоге выбери последовательный порт платы (CP2102 / CH340).
  4. Дождись окончания записи и перезагрузи плату.
